# RabbitMQ Example


This is an example of Rabbitmq with flask and Docker.


## How to use?

`create .env file and put email=YOUR EMAIL pass=YOUR PASS and PYTHONUNBUFFERED=0`

`docker-compose up -d`

edit and change email and message in `test.sh` to send mail.
